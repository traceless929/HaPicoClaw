# User Preferences

请把这个文件当作家庭环境偏好说明书。你可以在这里记录常用房间、设备别名、场景偏好和默认行为。

建议填写：

- 常用房间名称，例如客厅、主卧、书房、餐厅
- 常见设备别名，例如“主灯”对应哪个 `entity_id`
- 默认控制偏好，例如“夜间不要把亮度调到 100%”
- 需要额外确认的动作，例如门锁、窗帘、热水器

## 示例

- “客厅主灯”通常指 `light.living_room_main`
- “睡觉模式”优先执行 `scene.good_night`
- 晚上 23:00 后打开卧室灯时，优先使用较低亮度
- “关闭所有灯”默认只指卧室、客厅和餐厅，不包含阳台灯
- 涉及门锁、窗帘、热水器时，先确认再执行
